# 一个测试女友的考试系统

仅供娱乐，互相测验哦

## 启动项目

进入项目打开终端执行以下命令安装依赖包

### `npm start / yarn`

运行项目
### `npm run start / yarn start`

### 运行成功后打开 locaohost: 3000 即可开始考试
